# simple-regex-engine

支持的正则语法:

| 语法 | 描述 | 例子 | 匹配 |
|------|------|------|------|
| a | 匹配这个字符 | q | q |
| * | 匹配0个或者多个先前的字符 | a* | "", a, aa, aaa ... |
| ? | 匹配0个或1个先前的字符 | a? | "", a |
| . | 匹配任意的字符 | . | a, b, c, d ... |
| ^ | 匹配字符串的开始 | ^c | c, ca, cbb ... |
| $ | 匹配字符串的结尾 | a& | a, ba, cba ... |
